# Scientific-Programmer-Calculator-using-Tkinter
A modern Scientific and Programmer Calculator built using Python Tkinter.
This calculator provides basic arithmetic operations, scientific functions, number system conversions, and a real-time calculation history — all inside a clean, professional dark-themed GUI.

✨ Features
🔢 Basic Operations
Addition, Subtraction, Multiplication, Division
Power (^) and Square (x²)
Brackets support ()
Backspace and Clear functions
📐 Scientific Functions
Sine, Cosine, Tangent (Degree mode)
Square Root
Logarithm (base 10)
Factorial
Constants: π (pi) and e
💻 Programmer Functions
Convert Decimal → Binary
Convert Decimal → Hexadecimal
Convert Binary/Hex → Decimal
📜 History Panel
Displays all previous calculations
Helps track operations performed during the session
🎨 Modern UI Design
Dark professional theme
Segoe UI and Consolas fonts
Structured layout using Frames and Grid system
🖼️ Interface Preview

The application contains:

A large display screen for input/output
Calculator buttons arranged in grid format
A side history panel showing all calculations
🛠️ Technologies Used
Python 3
Tkinter for GUI
Math module for scientific calculations.
